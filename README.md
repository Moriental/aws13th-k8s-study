## Study Info
- **기간**: 4~5주
- **진행 방식**: 토픽별 담당자 배정 → 정리 → 발표 + 실습
- **참고 교재**: 쿠버네티스 완벽 가이드 (혹은 토픽을 웹 사이트 혹은 따로 정리하여 공부)
- **실습 환경**: 추후 결정

각자 본인 디렉토리에 토픽별 학습 내용과 실습 매니페스트를 정리합니다.

## How to Contribute
1. 본인 디렉토리에서 작업합니다.
2. 토픽별로 하위 디렉토리 또는 마크다운 파일을 생성하여 정리합니다.
3. 실습에 사용한 K8s 매니페스트 파일(.yaml)도 함께 저장합니다.
