### 내가 설정한 것
deployment의 컨테이너 부분에 포트 명시(kubectl edit deployment front-end -n sp-culator)
```
spec:
      containers:
      - image: nginx:1.14.2
        imagePullPolicy: IfNotPresent
        name: nginx
        ports:  -> 추가
        - containerPort: 80  -> 추가
          protocol: TCP  -> 추가
        resources: {} 
        terminationMessagePath: /dev/termination-log
        terminationMessagePolicy: File
```

노드 포트로 연결해주는 서비스 생성
```
kubectl expose deployment front-end --name=front-end-svc --port=80 --target-port=80 --type=NodePort -n sp-culator
```
