### 내가 설정한 것
가장 적합한 옵션 적용: netpol2.yaml
```
[root@k8s-master netpol]# cat netpol2.yaml 
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: netpol2
  namespace: backend
spec:
  podSelector:
    matchLabels:
      app: backend  -> backend 네임스페이스 중에서도 backend라는 태그를 가진 애들만
  policyTypes:
    - Ingress
  ingress:
    - from:
      - namespaceSelector:
          matchLabels:
            name: frontend  -> 출발지가 frontend 네임스페이스여야 함
        podSelector:
          matchLabels:
            app: frontend  -> 그중에서도 frontend라는 라벨을 단 애들이어야 함
```

netpol1.yaml: 제한이 없음
netpol3.yaml: backend 네임스페이스의 태그를 database로 잡음
