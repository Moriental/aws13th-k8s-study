### 내가 설정한 것
echo-sound라는 네임스페이스 안에 ingress라는 리소스 생성 및 적용
```
[root@k8s-master ~]# kubectl create ingress echo --rule="example.org/echo=echoserver-service:8080" -n echo-sound --dry-run=client -o yaml > ~/ingress.yaml
[root@k8s-master ~]# kubectl apply -f ~/ingress.yaml
```

---

### 결과
```
[root@k8s-master ~]# curl -o /dev/null -s -w "%{http_code}\n" http://example.org/echo --resolve example.org:80:192.168.56.40
200
```
