### 내가 설정한 것들
컴포넌트 조사
```
[root@k8s-master manifests]# cat /etc/kubernetes/manifests/kube-apiserver.yaml | grep etcd
    - --etcd-cafile=/etc/kubernetes/pki/etcd/ca.crt
    - --etcd-certfile=/etc/kubernetes/pki/apiserver-etcd-client.crt
    - --etcd-keyfile=/etc/kubernetes/pki/apiserver-etcd-client.key
    - --etcd-servers=https://192.168.56.41:2379  -> 마스터 노드가 웬 듣보잡 IP에 데이터를 읽는다고 하는 중
```

```
[root@k8s-master manifests]# ss -anp | grep 2379
```

kube-api.yaml 주소 수정
