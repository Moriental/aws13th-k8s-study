### 내가 한 설정들
PVC 파일 생성
```
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: mariadb
  namespace: mariadb
spec:
  accessModes:
    - ReadWriteOnce
  volumeMode: Filesystem
  resources:
    requests:
      storage: 250Mi
  storageClassName: local-path
```

deploy 파일 수정
```apiVersion: apps/v1
kind: Deployment
metadata:
  name: mariadb
  namespace: mariadb
spec:
  selector:
    matchLabels:
      app: mariadb
  template:
    metadata:
      labels:
        app: mariadb
    spec:
      containers:
        - name: mariadb
          image: busybox
          command: ['sh', '-c', 'echo "init data" > /var/lib/mysql/data.db && sleep 3600']
          volumeMounts:
            - name: mysql-persistent-storage
              mountPath: /var/lib/mysql
      volumes:
        - name: mysql-persistent-storage
          persistentVolumeClaim:
            claimName: mariadb ->빈 부분을 mariadb로 수정
```
---

### 트러블슈팅
1. 공식 홈페이지 양식 그대로 복붙해서 용량이 250Mi가 아닌 250Gi됨.
2. PVC에 스토리지 클래스가 있는 반면 PVC에는 스토리지 클래스를 명시하지 않아 매칭되지 않아 따로 추가.
