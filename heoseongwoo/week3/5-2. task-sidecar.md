1-2. 기존 컨테이너와 사이드카 컨테이너에 볼륨 연
```
containers:
      - name: synergy-app
        image: legacy-image
        volumeMounts:           
        - name: log-volume
          mountPath: /var/log

      - name: sidecar
        image: busybox:stable
        command: ["/bin/sh", "-c", "tail -n+1 -f /var/log/synergy-deployment.log"]
        volumeMounts:
        - name: log-volume
          mountPath: /var/log
```

## 새롭게 알게 된 점
legacy app: 과거에 만들어져서 지금까지 이용하고 있는 오래된 프로그램(시스템)
sidecar pattern: 옆에 보조 프로그램을 붙여 새로운 기능을 추가하는 것
InitContainer: 파드가 켜지자마자 초기화 컨테이너가 가장 먼저 되고 할 일을 마치고 사라지는 임시 컨테이너
